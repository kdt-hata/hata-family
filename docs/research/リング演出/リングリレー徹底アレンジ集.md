## 0. リングリレーとは？ ─ 古典の基本形

### 0-1. 古典形フロー
1. 1本の長いリボン（3〜10m）の両端を最後尾のゲストが持つ
2. リボンに指輪を通して、片端からもう片端へ「手から手へ」送る
3. 最終的に祭壇近くの新郎新婦（または立会人）に届く
4. 新郎新婦が指輪を交換する

### 0-2. なぜ感動するのか（メカニズム分解）
| 要素 | 効果 |
|---|---|
| 全ゲストが指輪に触れる | 「祝福が物理的に伝わる」体感 |
| リボンの長さで時間を作る | BGMと相まって「ためが効く」 |
| 一人ずつ目立つ | 自分の祝福が可視化される |
| 最後に新郎新婦の手に | クライマックスが明確 |

### 0-3. アレンジの軸（19方向）
本稿では以下の角度から派生案を出す。

```
リボン素材 → リボン本数 → 動線 → リレー対象 → ゲスト動作 → 音響 → 照明 →
ストーリー → テクノロジー → 文化融合 → ハロウィン → 屋上ドーム → 名前テーマ →
クライマックス → サプライズ → 多段階構成 → 子供/ペット → サステナブル → DIY
```
