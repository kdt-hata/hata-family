0用語整理 — 3階層

リング演出は3つの層に分けると整理しやすい。組み合わせて設計する。

①運び手 (Bearer)

指輪を祭壇まで「運ぶ」役割を担う主体

リングボーイ / リングドッグ / フクロウ / ドローン / ラジコン

②ヴェッセル (Vessel)

指輪を載せる物理的な台・道具

リングピロー / ボックス / 宝箱 / 貝殻 / ガラスドーム

③儀式 (Ritual)

指輪交換の前後に行う演出・祈り

リングウォーミング / リングリレー / エンゲージカバー

組み合わせ例: 「①リングボーイ × ②3本用ピロー × ③エンゲージカバーセレモニー」のように層を組み合わせると、唯一無二の演出になる。
