## 2. 判断軸の設計（なぜこの軸か）

「全部入りフラッグシップ」はどれも高機能なので、**“買ったあとに手がかからないか”と“暮らしの質を損なわないか”**で差がつきます。お二人のプロフィールから逆算して以下を設定し、太字を最優先にしました。

| # | 判断軸 | なぜ重要か（お二人にとって） |
|---|--------|------|
| ① | **手間ゼロ・全自動度** | 本当のコストは「掃除時間」でなく「ロボットの世話をする時間」。ゴミ/モップ/給排水/洗剤の自動化＋**立ち往生で呼ばれる回数の少なさ**（=AI回避） |
| ② | **静音性** | 芸術家の制作・経営者のオンライン会議。在宅で走らせても気にならないか（運転dB＋ゴミ収集音と時間指定可否） |
| ③ | デザイン/省スペース | ドックの存在感・本体の薄さ（家具下）。アーティスト宅に置ける佇まい |
| ④ | 清掃力 | 吸引(Pa)・モップ方式（加圧回転＝拭き取り力）・隅の取りこぼし |
| ⑤ | スマート連携 | Matter/Apple Home と Alexa/Google を**両方比較**。アプリ（マップ・部屋指定・進入禁止）の作り |
| ⑥ | プライバシー/データ | カメラ＋間取り＋クラウド。保存先・オンデバイス・メーカー姿勢・過去インシデント |
| ⑦ | ランニングコスト＆国内サポート | 消耗品費・保証・修理体制 |

## 4. 各機種の詳細

### 🥇 Roborock Saros 10R（第1推薦）
- **手間ゼロ**：8way全自動ドック＝80℃温水でモップ洗浄→**温風乾燥**（生乾き臭を回避）、自動給水、洗剤補充は約3か月に1回、ゴミ自動収集は**約60日**(2.7Lパック)。#1軸に直撃。
- **立ち往生の少なさ**：検証群で**AI回避スペックが最強**（3D ToF＋RGB／21,600点／108種認識／2cm以上を回避）。
  - ⚠️ ただしメーカー自ら「**100%回避は保証しない**」と明記。海外実機テストでは**黒いケーブルに絡まり救出が必要**だった例も → 「呼び出しは減るがゼロではない」。
- **デザイン**：本体高さ **7.98cm で検証群最薄**（家具下に入りやすい）。ドック 381×475×488mm。
- **プライバシー**：「**画像を一切保存しない設計**」＋TÜV Rheinland認証＋Matter（※ベンダー表明であり第三者監査ではない点は割引いて評価）。
- **連携**：Alexa/Google/Siri＋Matter。**Apple Homeは現状未対応**で今後のOTA予定（国内ページ表記）。
- 価格：希望小売 ¥269,800 → 実勢 **約¥159,800**。

## 5. 🚨 買う前に必ず知っておくべき “落とし穴” 3つ

1. **「自動給排水」は全機種、標準ドックでは実現しない。** 標準は手動の清水/汚水タンク。水道直結の自動給水＋自動排水には**別売キット**（Roborock「Refill and Drainage System」、Dreame「Water Hookup Kit RAW7」）が必要。自動の“**洗剤補充**”はRoborock/Dreame/Ecovacsで満たすが、自動“**排水**”は全機種で追加費用＋設置工事。→ **新居の洗面/ランドリーに給排水を引けるか**が鍵。
2. **静音(②軸)の実数値が、第1・第2推薦ではまだ未確認。** 検証で確証できた運転音は Ecovacs 65dB(うるさい) と Eufy 54.7–64dB のみ。**Roborock/Dreame の運転モード別dBは確証データに無い**＝購入前の最後の確認ポイント。一般にフラッグシップは標準58–62dB級だが、実機/レビューで詰めたい。
3. **Narwalの静音・AI数値は検証で棄却。** マーケ資料の53dB・回避スペックは当てにしない。

## 6. 🔐 プライバシー（AI経営者の観点）

カメラ＋間取り＋クラウドは **“どのメーカーも一度は燃えている”前提**で選ぶのが現実的：
- **Ecovacs**：2024年に研究者からハッキングで盗撮・盗聴の可能性を指摘（[TechCrunch](https://techcrunch.com/2024/08/09/ecovacs-home-robots-can-be-hacked-to-spy-on-their-owners-researchers-say/)）。
- **iRobot/Roomba**：開発機の室内画像が外部流出した過去（[MIT Tech Review](https://www.technologyreview.com/2023/01/26/1067317/podcast-roomba-irobot-robot-vacuums-artificial-intelligence-training-data-privacy-consent-agreement-misled/)）。
- **Eufy**：2023年にクラウド暗号化を巡る問題。

→ その点で **Roborockの「画像を保存しない設計＋TÜV認証」表明は方向性として好材料**（第三者監査ではない点は割引いて）。寝室にカメラを置くなら、**進入禁止エリア**やカメラ無効化の運用も併用を。

## 7. お二人のケースへの当てはめ／最終提案

1. **フローリング中心・カーペット/段差なし**：カーペット持ち上げや段差越えの優先度は低く、**全自動度・静音・薄さ・拭き上がり**で選んでよい条件。
2. **とても忙しい**：①手間ゼロが最重要 → ゴミ60日級＋モップ自動洗浄/温風乾燥＋AI回避でRoborockかDreame。
3. **在宅で制作・会議**：②静音重視 → **Ecovacs(65dB)は避けたい**。
4. **AI当事者**：プライバシー表明とアプデ姿勢を重視 → Roborockの方向性が合う。

**最終提案：**
- 🥇 **Roborock Saros 10R**：全自動・AI回避・薄さ・プライバシーのバランスが最良。迷ったらこれ。
- 🥈 **Dreame X50 Ultra**：ダスト100日とApple Home稼働を取るなら。手間ゼロはほぼ同等。
- 🧽 **Ecovacs X8 PRO OMNI**：拭き取り最優先で、静音の妥協を許せる場合だけ。
- 🛟 **SwitchBot**：スペック互角で**国内サポート最重視**なら検討（最上位機の実力は要追加調査）。

## 8. 注意点・次のアクション

- [ ] **静音dBの実測を確認**：Roborock Saros 10R / Dreame X50 Ultra の運転モード別dBとゴミ収集音、**収集時刻のスケジュール可否**（②軸の最終確認）。
- [ ] **新居の給排水**：自動給排水キット（Dreame RAW7 / Roborock Refill&Drainage）の価格と、新築マンションで現実的に設置できるか。
- [ ] **実勢価格を価格.comで再確認**：MSRPから大きく下落中（Saros 10R ¥269,800→約¥159,800 等）。
- [ ] **消耗品の国内価格と交換周期**（モップ/フィルター/紙パック/洗剤）の年間ランニングコスト。
- [ ] 候補を2つに絞ったら、可能なら実機の動作音と立ち往生の癖を確認。
