# 結婚式 Web招待状サービス比較（無料中心）

> 作成日：2026-05-28 / 対象：畑昂希 ✕ 玉置ひかり 結婚式（会場：AT TERRACE、挙式 2026-10-31、二次会を検討中）
> AT TERRACE さんから「利用者が多い」と紹介された3サービス（PIARY / BRAPLA / Weddingday）を中心に、他の有力サービスも加えて調査・比較しました。
> ※料金・仕様は調査時点（2026年5月）のもの。**申込み前に必ず各公式サイトで最新情報をご確認ください**（特に手数料は変動・情報源差があります。後述の「注意点」参照）。

## 1. 結論（先に要点だけ）

- **大前提：紹介された3つはどれも「招待状づくり〜出欠管理」までは完全無料**。差が出るのは ①デザイン ②ご祝儀・会費の「オンライン決済」を使うかどうか・その手数料 ③付随機能、の3点だけです。
- **「招待状＋出欠管理だけ使う（ご祝儀はオンライン決済しない）」なら → どれを選んでも実質タダ。好みのデザインで決めてOK。**
  - デザイン数・統一感（席次表まで無料）重視なら **BRAPLA**（テンプレ業界最多＋WEB席次表も無料）。
  - 広告ゼロ・写真/動画多め・多言語なら **Weddingday**。
- **二次会（会費制）や、ご祝儀をオンラインで事前に集めたいなら → 手数料の安さが効いてくる。**
  - 金額が大きいほど **定額3,500円の「楽々!WEB招待状」** か、**銀行振込0.9%の Weddingday** が圧倒的に有利。
  - PIARY(3%) / BRAPLA(3.1%) は%課金なので、集金額が大きいと割高になります。

**👉 総合おすすめ：**
- **メインの招待状は `BRAPLA`（無料の完成度が高くデザイン最多・席次表も無料）** をベースに、
- **二次会の会費 or ご祝儀をオンラインで集めるなら、その用途だけ `楽々!WEB招待状`（定額3,500円）か `Weddingday`（振込0.9%）を併用**する、のがコスパ最適です。
- 「とにかく1つで完結させたい・準備物との連携も欲しい」なら **PIARY**（席次表・席札・引き出物宅配まで情報連携）も堅実な選択。

## 2. そもそも「良い」とは？──判断軸の設計

Web招待状は「無料」を謳うサービスがほとんどです。なので「無料かどうか」では差がつきません。**“無料の範囲の広さ”と“有料になる一線（＝主にご祝儀オンライン決済の手数料）”、そして機能・デザイン**で見るのが正解です。以下の軸で評価しました。

| # | 判断軸 | なぜ重要か |
|---|--------|-----------|
| A | **基本機能の料金（作成・送付・出欠管理）** | ここが無料かどうか。3社とも無料だが上限（作成数など）に差がある |
| B | **ご祝儀・会費のオンライン決済の手数料** | 唯一の主要な「有料ポイント」。%課金か定額か、誰が負担するかで実費が大きく変わる |
| C | **デザイン数・テイスト** | 招待状の第一印象。前撮り写真を活かせるか |
| D | **出欠管理（RSVP）** | 名簿の自動化・CSV出力・送り分け・リマインダーの有無 |
| E | **写真・動画** | 掲載枚数、メッセージ動画やカウントダウン演出 |
| F | **アンケート/カスタム質問** | アレルギー確認・自由質問・ゲストからのメッセージ回収 |
| G | **二次会・1.5次会対応** | ← 今回二次会を検討中なので重要。会費制への対応 |
| H | **準備物との連携** | 席次表・席札・引き出物などへ名簿を引き継げるか |
| I | **その他（多言語・広告有無・保存期間・サポート・会場持込実績）** | 細かいが満足度に効く |

> 補足：ご祝儀（披露宴）は伝統的に「当日ご祝儀袋で手渡し」が主流で、オンライン決済を使わないカップルも多いです。一方 **二次会・1.5次会の「会費制」はオンライン事前決済と相性が良い**（受付がラク・未払い防止）。今回は二次会検討中なので、軸B（手数料）は会費用途で効いてきます。

## 3. 料金の全体像（無料 / 有料の境界線）

```
┌─────────────────────────────────────────────┐
│ 【完全無料ゾーン】どのサービスも基本ここまでタダ        │
│  ・招待状の作成（テンプレ／写真／動画）                │
│  ・LINE/メール/SNS/QRでの送付（何通でも）            │
│  ・出欠管理（RSVP）・名簿の自動作成・CSVダウンロード      │
│  ・送り分け／リマインド／アンケート／カウントダウン 等     │
├─────────────────────────────────────────────┤
│ 【ここから有料】＝実質「ご祝儀・会費のオンライン決済」だけ  │
│  ・ゲストがクレカ等で事前に祝儀/会費を払える機能         │
│  ・手数料が発生（%課金 or 定額。サービスごとに差大）       │
│  ・※紙併用や一部の特殊オプションが有料の場合もあり        │
└─────────────────────────────────────────────┘
```

**つまり「無料でどこまで？」の答え＝招待状づくりと出欠管理は全部無料。「有料は何？」の答え＝ご祝儀・会費をオンライン決済する時の手数料のみ。** ここを理解すれば選択はシンプルになります。

## 6. 各サービスの詳細メモ

### PIARY（ピアリー）
- 招待状は完全無料。**最大の魅力は“準備物への一気通貫”**＝集めたゲスト名簿を席次表・席札・引き出物宅配「ヒキタク」へそのまま連携でき、後工程がラク。
- デザイン150種以上、写真27枚、メッセージ動画・カウントダウン対応。LINE専門スタッフのサポートあり。
- 決済手数料は3%+660円（%課金）。**手数料負担を新郎新婦/ゲストで選べる**のは地味に便利。
- 先輩花嫁の利用率2割前後と実績十分。紙招待状との統一もしやすい。

### BRAPLA（ブラプラ）
- **テンプレ業界最多クラス（200〜242種）**でデザインの幅が圧倒的。前撮り写真も映える。
- **WEB席次表まで無料**で、招待状と世界観を統一できる。**付箋機能**（受付・余興など個別案内）が独自で便利。
- 招待状は**いくつでも無料・データ無期限保存**。同伴者登録、アレルギー登録、リマインドメールなど管理機能が充実。
- ユーザー満足度98.2%を掲げ、全国の会場で持込実績あり。決済は3.1%（%課金）。

### Weddingday（ウェディングデイ）
- **写真40枚＋動画・40言語対応・ゲストへの広告ゼロ**と、ゲスト体験の質が高い。海外/外国人ゲストがいるなら特に有利。
- **オンライン決済が銀行振込0.9%と最安水準**（ただしクレカは3.9%）。会場によっては手数料を会場が負担してくれるケースも。
- 作成は最大5種類（挙式・二次会など場面別に作れる）。合言葉で閲覧制限可、CSV一括DL対応。
- カスタムで実質2,000パターン以上。利用者数No.1を掲げる定番。

### （参考）楽々!WEB招待状
- 今回は会場紹介に含まれていないが、**ご祝儀/会費の決済手数料が「一律3,500円」と定額で、集金額が大きいほど最安**。二次会会費やご祝儀オンライン化を本気で考えるなら有力。
- デザイン100種以上、AIプランナーによる文面添削あり。基本機能は無料。

### （参考）その他
- **Favori**：別ランキングで1位の評価。デザイン性に定評。
- **Biluce（5%）／Dear／KITENE／Concept Marry（提携会場のみ・定額4,980円）** など多数。手数料はおおむね高め。

## 7. お二人のケース（AT TERRACE / 10-31挙式 / 二次会検討）への当てはめ

1. **時間は十分**：挙式まで約5か月。Web招待状は短期間で作れるため、デザイン選びにじっくり時間を使える。
2. **会場の持込**：3社とも全国の会場で持込実績があり、AT TERRACE で問題になる可能性は低い（念のため会場へ一言確認すると安心）。
3. **二次会（会費制）を本当にやるなら**：受付の手間と未払い防止のため**オンライン会費決済**が効く → その用途は **楽々（定額3,500円）** か **Weddingday（振込0.9%）** が手数料的に有利。
4. **ご祝儀（披露宴）**：当日手渡しにするなら決済手数料は不要＝デザイン重視で選んでOK。

**最終提案：**
- 🥇 **メイン招待状＝BRAPLA**：無料の完成度が高い（デザイン最多／席次表も無料／付箋・同伴者・リマインド充実）。コストゼロで満足度が高く、迷ったらこれ。
- 🥈 **PIARY**：席次表・席札・引き出物まで“1か所で完結”したい効率重視派に。
- 🥉 **Weddingday**：写真をたくさん載せたい／外国人ゲストがいる／広告を一切見せたくない／二次会会費を振込0.9%で集めたい場合。
- 💰 **二次会会費 or ご祝儀のオンライン決済を使うなら**：その部分だけ **楽々!WEB招待状（定額3,500円）** を併用すると手数料を最小化できる（メインの招待状と別サービスでも問題なし）。

## 8. 注意点・次のアクション

- [ ] **手数料の最新値を公式で確認**：特に Weddingday（公式0.9%〜 vs 比較記事4%+260円）と BRAPLA の振込手数料有無。決済を使う場合は申込み前に要確認。
- [ ] **オンライン決済を使うか先に決める**：使わない（当日手渡し）なら手数料軸は無視でき、純粋にデザイン・機能で選べる。
- [ ] **AT TERRACE に持込可否を一応確認**（紹介3社なのでほぼ問題なしのはず）。
- [ ] **二次会の実施可否が固まってから**、会費オンライン化の要否＝楽々/Weddingday併用の要否を判断。
- [ ] 候補を2つほどに絞ったら、実際にテンプレを触って**スマホでの見え方**を二人で確認（無料で試せる）。
