## 構成ルール

- 1トピック = 1フォルダ（例: `招待状/`, `会場/`, `二次会/` …）
- 各フォルダに調査結果を Markdown で置き、必要に応じて見やすい HTML 版も併置する
  - 例: `Web招待状サービス比較.md`（原本） / `Web招待状サービス比較.html`（閲覧用）
- このREADMEの「調査一覧」表に1行追加して全体を俯瞰できるようにする
