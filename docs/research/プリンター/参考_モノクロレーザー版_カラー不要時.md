## 2. なぜ「モノクロレーザー」なのか（インクジェットとの比較）

書類メイン＋ランニングコスト重視という条件では、家庭用でも**レーザーが理にかなっています**。

| 観点 | モノクロレーザー（推奨） | インクジェット |
|---|---|---|
| 書類の印刷品質 | 文字がにじまず**即乾**。マーカー・水濡れに強い | 普通紙だと文字がにじむ場合あり |
| 放置耐性 | 数週間使わなくても問題なし | **長期放置でノズル目詰まり** → 復旧クリーニングでインクを浪費 |
| 1枚あたり(モノクロ) | 約3.3〜4.5円（推奨機・大容量トナー時） | 約2.2円（※特定の大容量インク機の楽観値） |
| 色 | **モノクロのみ（カラー不可）** | カラー印刷できる |
| 速度 | 速い（29枚/分級） | 遅め |

⚠️ **“数字のワナ”に注意**：「インクジェットはモノクロ約2.2円で最安」という比較をよく見かけますが（ASKUL試算）、これは**特定の大容量インク機種（ブラザー MFC-J7500CDW-A）ベースの楽観値**で、一般的な家庭用インクジェットには当てはまりません。
逆に「レーザーは一般にランニングコストが安い（ただし電気代は高い）」という“通説”も、今回の敵対的検証では**裏が取れず棄却**しました（出典の主張を確証できなかったため断定しない）。
→ なので本レポートは**通説ではなく、推奨機の実数値（3.3〜4.5円/枚）**で判断しています。

**結論：色が要らない書類用途なら、トナー寿命・即乾・放置耐性の実用メリットでレーザーが有利。** ただし「カラーが一切出ない」点だけは許容が必要（後述の落とし穴①）。

## 5. 🍎 Apple機器からの印刷（マスト要件の検証）

**結論：4機種すべて、iPhone / iPad / MacBook から “追加アプリなし” で無線印刷OK。** ここは全候補が合格で、差はつきません。

- **iPhone / iPad**：標準のAirPrintフロー。Safari等でページを開き **共有ボタン → プリント → プリンタを選択 → プリント**。専用アプリもドライバも不要（Apple公式・ブラザー/キヤノン公式が同一手順を明記）。
- **MacBook**：同じWi-Fiに繋いで「プリンタを追加」するだけ。特にブラザーは **macOS用ドライバを提供していない** ため、**AirPrintが正規かつ標準の印刷経路**＝余計なドライバ管理が要らず、むしろシンプル。
- **セットアップの簡単さ**：いずれも同一Wi-Fiに参加させればすぐ使える。初期設定の容易さでは横並びで優秀。

## 6. 🚨 買う前に知っておきたい “落とし穴”

1. **モノクロレーザーは「色が一切出ない」。** 書類専用と割り切る前提。**アーティストの色校正・カラー資料・年賀状の写真面**などが必要になったら、別途インクジェット or コンビニ/ネットプリントで対応する想定に。お二人は「写真は重視しない／書類メイン」なので基本問題なし、という判断。
2. **「3.3円/枚」は条件付きの数字。** ①**大容量トナー**使用時 ②**印字率5%（ISO規格）** ③**トナーのみ**の値。**ドラムユニット込みなら約4.0円/枚**。さらに**本体同梱の初期トナーは少量**（ブラザー同系機で約700枚程度）なので、早めに大容量トナーの買い足しが要る点も予算に入れておく。
3. **複合機（DCP-L2600DW / MF272dw）は厳密には「単機能でOK」条件から外れる。** ただし**ほぼ同価格でスキャン/コピーが付く**ので、むしろお得なケースも。「省スペース最優先で印刷だけ」なら単機能（HL-L2460DW / LBP122）を選ぶ。
4. **価格は日々変動。** 各機種 ¥11,000〜¥13,900 のレンジで動く。購入直前に価格.com / 各ストアで再確認を。
5. **設置スペース・重量・動作音は要現物確認。** 一般にレーザーはインクジェットより筐体が大きめ・重め。新居の置き場所（棚の耐荷重・奥行き）を測っておくと安心（※本調査では各機の正確な寸法・騒音は未確認）。

## 7. お二人のケースへの当てはめ／最終提案

1. **書類メイン・写真は重視しない** → カラー不要なので**モノクロレーザーが最適**。色のデメリットを受け入れられる条件。
2. **とにかく無線でサッと（マスト）** → 全候補がAirPrint標準対応で**追加アプリ不要**。差がつかないので、ここは安心して他軸で選べる。
3. **本体安さ＋ランニング重視** → 本体もトナー単価も最安級の **HL-L2460DW** が二冠。次点LBP122は本体・ランニングとも少し高い。
4. **5GHz Wi-Fi** → マンションの電波混雑に強いのは **HL-L2460DW / DCP-L2600DW**（どちらも5GHz対応）。

**最終提案：**
- 🥇 **ブラザー HL-L2460DW**：迷ったらこれ。単機能・最安・3.3円・AirPrint・自動両面・5GHz とお二人の条件を全部満たす。
- 🔁 **ブラザー DCP-L2600DW**：「同じ値段ならスキャン/コピーも」なら本命の上位互換的に。新婚世帯は役所/契約書類のスキャンも地味に便利。
- 🥈 **キヤノン Satera LBP122**：完全な単機能の純度・キヤノン党なら。コストは一歩譲る。

> **一言でいうと：** 印刷だけで省スペース重視なら **HL-L2460DW**、ほぼ同額でスキャンも欲しいなら **DCP-L2600DW**。この2択でほぼ正解です。

## 8. 注意点・次のアクション

- [ ] **印刷枚数の見積もり**：月あたり何枚刷るか。多い（月300枚以上）なら単価差が効くので3.3円機を、少ないなら本体価格優先でOK。
- [ ] **単機能 or 複合機の最終判断**：スキャン/コピーを使うか？ → 使うなら DCP-L2600DW、使わない＆省スペースなら HL-L2460DW。
- [ ] **実勢価格を価格.comで再確認**（¥11,000〜¥13,900で変動）。
- [ ] **純正トナーの実売価格と交換周期**（TN-29J / TN32JXXL / CRG-071H 等）を確認。互換トナーは安いが保証/故障リスクがあるため本レポートは純正前提。
- [ ] **置き場所の寸法・耐荷重・動作音**を現物/レビューで確認（レーザーはやや大きめ）。
- [ ] （任意）**カラーが必要な場面**の有無を最終確認。必要ならコンビニ/ネットプリント運用 or カラー機の併用を検討。
